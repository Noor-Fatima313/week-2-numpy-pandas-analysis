# Dataset Cleaning and Exploratory Analysis using Pandas

## Project Overview

This project focuses on cleaning, exploring, and analyzing the Titanic dataset using **NumPy** and **Pandas**. The objective is to understand the dataset, handle missing values, perform data manipulation, and extract meaningful insights through exploratory data analysis (EDA).

## Dataset

* **Dataset:** Titanic Dataset
* **Source:** Kaggle
* **Format:** CSV

## Libraries Used

* Python
* NumPy
* Pandas
* Jupyter Notebook / Google Colab

## Steps Performed

* Loaded the dataset using Pandas.
* Explored the dataset structure and summary statistics.
* Identified missing values and duplicate records.
* Cleaned the dataset by handling missing values and removing duplicates.
* Renamed columns and converted data types where needed.
* Performed filtering, sorting, grouping, aggregation, and feature creation.
* Exported the cleaned dataset.

## Key Insights

* Female passengers had a higher survival rate than male passengers.
* Most passengers traveled in Passenger Class 3.
* The `Cabin` column contained the most missing values.
* Higher fares were generally associated with higher passenger classes.
* Most passengers traveled alone or with small families.
